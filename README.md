# seo-geo

SEO & GEO (Generative Engine Optimization) for websites. Analyze keywords, generate schema markup, optimize for AI search engines (ChatGPT, Perplexity, Gemini, Copilot, Claude) and traditional search (Google, Bing). Use when user wants to improve search visibility, search optimization, search ranking, AI visibility, ChatGPT ranking, Google AI Overv

## Install

```bash
npx skills add https://github.com/Aston1690/seo-geo --yes
```

## Usage

After installing, this skill will be available in Claude Code and other compatible AI coding agents.
